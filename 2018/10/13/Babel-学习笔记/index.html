Babel 学习笔记

发表于 2018-10-13 | 分类于 工具 |

Babel 通过语法转换器支持最新版本的 JavaScript。 这些插件允许你立刻使用新语法，无需等待浏览器支持。

基本使用

  1. 新建项目目录，初始化 npm

  2. 在项目根目录下安装 Babel

    每年每个 preset (es2015) 只编译当年批准的内容。 而 babel-preset-env 相当于 es2015 ，es2016 ，es2017 及最新版本。

  3. 安装 babel-polyfill

    由于 Babel 只转换语法(如箭头函数)，不转换新的 API。你可以使用 babel-polyfill 支持新的 API，例如全局变量 Promise Set Map 等 、原生方法如 String.padStart (left-pad) 等。

    使用它时需要在你应用程序的入口点顶部或打包配置中引入 babel-polyfill

  4. 在项目根目录下创建 .babelrc 配置文件

  5. 修改 package.json 文件

  6. 在命令行使用 Babel 进行转码

常用指令

# babel
